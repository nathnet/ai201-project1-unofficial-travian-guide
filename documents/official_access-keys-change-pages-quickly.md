# Access Keys - change pages quickly

> Source: Travian: Legends Support  
> URL: https://support.travian.com/en/articles/80-access-keys-change-pages-quickly

---

Access keys allow you to **navigate between game pages faster** using your keyboard.
This feature works in **all major browsers**, though the key combinations may vary slightly depending on which one you use.

---

### Using Access Keys

Access keys are shortcuts that take you directly to key sections of *Travian: Legends* without clicking through menus.

---

### Travian: Legends Access Keys

| **Access key** | **Page which opens with the access key** |
| --- | --- |
| 1 | Resources |
| 2 | Buildings |
| 3 | Map |
| 4 | Statistics |
| 5 | Reports |
| 6 | Messages |
| 7 | Daily Quests |
| 8 | Gold Shop |
| 9 | Village overview |
| b | Previous village (goes to the previous village in the village list) |
| n | Next village (goes to the next village in the village list) |

---

### Tips

- These shortcuts work best when playing in **fullscreen** or **dedicated browser windows**.
- If the shortcut doesn’t work, ensure **no browser extension or system shortcut** is overriding it.
- You can use **b** and **n** to quickly switch between villages during active gameplay or raiding sessions.
