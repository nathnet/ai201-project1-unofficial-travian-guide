# Glossary

> Source: Travian: Legends Support  
> URL: https://support.travian.com/en/articles/167-glossary

---

## Symbols & Numbers

- **0 pop (zero)** — A player or village with no inhabitants.
- **18c / 15c / 9c / 7c / 6c** — Tile with 18/15/9/7/6 crop fields.
- **+/+ , +/- , -/+ , -/-** — NE, SE, NW, SW map quadrants.

---

## A

- **Aca** — Academy.
- **Afk** — Away from keyboard; often used to mean inactive.
- **Aggro** — Alliance focused on fighting over WW play.
- **Aka** — Also known as.
- **Ally** — Alliance.
- **Analyser** — External stats/report sites (e.g., inactive search, Travco, Gettertools).
- **Anvil** — Large defensive army.
- **Artefact** — Special bonus item (activated in treasury on special worlds).
- **Asap** — As soon as possible.
- **Axes** — Axeman.

## B

- **Bashing** — Keeping a player small via frequent attacks.
- **Battle groups** — Sub-teams acting together for ops/defense.
- **Bonus (oasis)** — Resource bonus from oases.
- **Boonies** — Far from (0|0); fewer close fights, focus on growth.
- **BP** — Beginner’s Protection; also Building/Construction Plans (context).
- **Brb / Btw** — Be right back / By the way.
- **Bug** — Game error.

## C

- **C** — “See.”
- **Caesaris** — Equites Caesaris.
- **Capital** — Only one; fields >10; cannot be conquered (can be moved).
- **Cata / Catapults** — Destroy buildings; count as infantry.
- **CC** — Command Centre or Corinthian Crushers.
- **Cheating** — Rule violations.
- **Chief** — Unit that reduces loyalty to conquer (generic name).
- **Chiefing** — Conquering by lowering loyalty.
- **Clear / Cleaner** — Army/wave to remove defense before catas/chief.
- **Clubbies** — Clubswingers (Teuton t1).
- **Cluster** — Own villages near each other; grouped for protection.
- **CM** — Community Manager.
- **Confed** — Confederacy (up to three alliances).
- **Construction Plans** — Required to build WW; second set needed past lvl 49.
- **Coords** — Coordinates.
- **Counter** — Follow an enemy army back home to hit it.
- **CP** — Culture Points; also Construction Plans (context).
- **Croplocked** — Village forced deeply negative in crop.
- **Crop / Cropper** — Food resource; a 9c/15c (18c on specials).
- **Cya / Cyt** — See you / See you tomorrow.

## D

- **Def** — Defensive troops.
- **Deletion** — Avatar removal; takes 72 hours.
- **Dmg** — Damage.
- **Double Target** — Catas hit two buildings (Rally Point 20).
- **Druids** — Druidrider.
- **Dual / Dual Construction** — Multiple people on one avatar; Romans build field+building at once.

## E

- **EC / EI** — Equites Caesaris / Equites Imperatoris.
- **Elite** — Single alliance, no wings/confeds (opposite of meta).
- **End Game / Hammer / Meta** — Post-plan phase; hammer for plans/WW sites; meta supporting WWs.
- **Epic / Exp / Expansion slot** — Awesome; hero XP; slots in palace/residence/command centre.

## F

- **Fake** — 20-unit fake to hide true target.
- **Farm / Farmer / Farming** — Raided account; person who raids; repeated raiding (incl. oases).
- **FTW** — For the win.

## G

- **GB / GG / GS / GW** — Great Barracks / Granary / Stable / Warehouse.
- **GL / GN / GZ** — Good luck / Good night / Congrats.
- **Ghost hammer / Ghosting** — Fast cav hammer; killing enemy off at home.
- **Golding** — Using gold.
- **GTG** — Got to go.
- **Guides** — Official guide index.

## H

- **Haeds** — Haeduan.
- **Hammer / Hammer village** — Large off army; village that builds it.
- **HDT** — Horse Drinking Trough.
- **Hero** — Your avatar’s unit with stats/bonuses.

## I

- **IGM / IGN** — In-game message / name.
- **IIRC / IMO** — If I remember correctly / In my opinion.
- **Imp** — Imperian.
- **Insert** — See “Sniping”; also offensive villages placed close to enemy for short-range fights.

## J / K / L

- **JK** — Just kidding.
- **Kick** — Remove from chat/alliance.
- **L8r / Lag / Leg / Loot / Lvl** — Later; delay; Legionnaire; stolen resources; level.

## M

- **Macemen** — See Clubbies. **MB** — Main Building.
- **Meta** — Group of alliances working together. **MH** — Multihunter (Rule Enforcement).
- **Mill** — Grain Mill. **Mirror kill** — See Second Kill. **MM** — Mass Message. **MP** — Marketplace. **Multi** — Multiple accounts (illegal).

## N

- **NAP** — Non-Aggression Pact. **Newbie / Noob** — New player / derogatory for newbie. **NP** — No problem. **NPC / NPCing** — Exchange merchant / using it.

## O

- **Oasis** — Conquerable map spot that grants production bonus; requires Hero’s Mansion. **Off** — Offensive troops.

## P

- **PA** — Plus Account. **Pala / Pally** — Palace or Paladin (Teuton).
- **Parking village** — Maximizes crop to feed stationed troops.
- **Phala** — Phalanx. **Plus** — Plus Account. **PM** — Private message. **Pop** — Population.
- **Pushing / Pusher** — Sending resources for a purpose / the player who does it.

## Q

- **Quad / Quad-line** — Quadrant and its divider line. **Quests** — Daily Quests / Tasks.

## R

- **R / RL / ROFL** — Are / Real life / Rolling on the floor laughing.
- **Raid / Raider** — Attack to steal resources / the player who raids.
- **Ram / Rammer** — Wall-destroying unit / ram hammer.
- **Recruiter** — Finds new members. **Reinforcements (Reins)** — Troops stationed to defend.
- **Res / Resi** — Resources / Residence. **ROI** — Return on investment (see guide). **RP** — Rally Point.

## S

- **Script** — Third-party modification; illegal.
- **Scouts / Scout hammer or anvil** — Recon unit; many scouts for offense/defense.
- **Second Kill / Secondary Clear** — Landing a second later to kill; second clearing wave.
- **Server (gameworld)** — One round of Travian. **Sim City / Simmer** — Pop-first playstyle / player.
- **Sitter / Sitting** — Playing via sitter access.
- **Spam / Spammer** — Off-topic post / person.
- **Speed server** — x2/x3/x5/x10 speed vs x1.
- **Spike** — Hide defense in a farmed village to hurt raiders.
- **Spy (troop / player)** — Scout unit / infiltrator. **Split** — See Sniping. **Sniping** — Insert defense between waves.
- **Starvation** — Troops die when crop is zero. **Supporters** — Support staff. **Swords** — Swordsman.

## T / U / V

- **TC / TK / TO / TS / TT** — Treasure Chamber / Teutonic Knight / Trade Office / Tournament Square / Theutates Thunder.
- **U / Ur** — You / you’re or your.
- **(V)** — (No common entries listed.)

## W

- **Wall** — City/Earth Wall/Palisade; boosts defense.
- **Waves / Wb** — Multi-attack timing tactic; Wave Builder tool.
- **Wings** — Allied side-alliances. **WW / WWK** — World Wonder / WW killer hammer.

## X / Y / Z

- **(X, Y)** — (No common entries listed.)
- **Zerg Meta** — “Many members” meta beyond confed limits. **Zero** — See 0 pop.
