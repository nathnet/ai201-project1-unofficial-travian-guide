# Alliance Management

> Source: Travian: Legends Support  
> URL: https://support.travian.com/en/articles/86-alliance-management

---

Alliance management tools allow alliance leaders and authorized members to organize, communicate, and coordinate their alliance effectively. This article explains which actions are available, which permissions are required, and how common alliance management features work. It is intended to help alliance leaders and officers use these tools correctly without needing customer support.

---

## Alliance Management Rights

Alliance management actions are only available to players who have been granted the appropriate rights. By default, the **alliance founder** has full permissions. Additional rights can be assigned to other members.

Alliance leaders and authorized members can:

- Change the alliance name and tag
- Assign or revoke alliance rights
- Monitor member activity status
- Edit the alliance profile and description
- Manage diplomacy information
- Assign member specializations
- Send mass messages (MMs)

Rights are managed from the **Embassy → Alliance → Options → Assign to position** menu.

---

## Changing Alliance Name and Tag

Players with the required permissions can change the alliance name and tag:

1. Open the **Alliance** interface
2. Go to **Options**
3. Edit the **Tag** and **Name** fields
4. Save the changes

The existing name and tag will be shown and can be overwritten directly.

---

## Assigning Alliance Rights

Alliance rights can be granted to members by players who already have permission to assign roles.

To assign rights:

1. Open **Embassy → Alliance → Options**
2. Select **Assign to position**
3. Find the player in the member list
4. Select the rights to grant
5. Save the changes

### Important Note on Permissions

Be cautious when granting the **Assign to position** right. Any player with this permission can change or remove your own rights, including those of the alliance founder.

---

## Available Alliance Rights

Depending on permissions granted, alliance members may be able to:

- Change the alliance description
- Manage alliance diplomacy display
- Send mass messages (MMs)
- Invite players to the alliance
- Kick players from the alliance
- Manage the alliance forum
- Manage map flags and markers
- Assign member specializations

---

## Member Activity Status

Alliance member lists display colored activity indicators that show recent player activity:

- **Blue**: Player is currently active
- **Green**: Active within the last 24 hours
- **Yellow**: Active within the last 3 days
- **Red**: Active within the last 7 days
- **Grey**: Inactive for more than 7 days

Players who remain inactive for an extended period may have their account automatically deleted by the game.

---

## Alliance Profile and Description

Each alliance has a public profile that can be edited by authorized members.

The alliance description consists of **two separate text fields**:

- Text entered in the **left field** appears on the left side of the profile
- Text entered in the **right field** appears on the right side of the profile

This area is commonly used to describe:

- Alliance goals
- Recruitment requirements
- Internal rules
- General alliance information

### Displaying Diplomacy Automatically

To automatically display alliance relations, you can use the following tags in the description:

- `[diplomatie]` – displays all relations
- `[ally]` – shows allied alliances
- `[nap]` – shows non-aggression pacts
- `[war]` – shows wars

These tags can be combined or used individually.

---

## Member Specializations

Players with the appropriate rights can assign **specializations** to alliance members:

- Offensive
- Defensive

This is done via **Alliance Profile → Members list**.

Specializations serve two purposes:

- Visually marking player focus (offense or defense)
- Allowing targeted mass messages to specific groups

---

## Mass Messages (MMs)

Mass messages allow alliance-wide or group-specific communication. Only players with the appropriate rights can send them. The alliance founder has this permission by default.

### Sending Mass Messages Using Commands

When writing an in-game message, you can use the following commands as the recipient (including brackets):

- `[ally]` – sends the message to all alliance members
- `[off]` – sends the message to members marked as offensive
- `[def]` – sends the message to members marked as defensive

Alternatively, you can use the **alliance button** next to the recipient field, which automatically inserts `[ally]`.
