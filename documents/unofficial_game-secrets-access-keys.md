In a web browser, an access key allows a computer user to immediately jump to a specific web page via the keyboard.  Not many people know, but pressing a few keys on the keyboard allows quickly go to a specific page in game.

All major browser support this feature. It might be activated in a slightly different way in various browsers, yet, in most cases it is a combination of**ALT + Access key** or **CTRL + Access key**. If neither of those options work for you, you can look up the needed access key combination in  [Wikipedia article about access keys.](https://en.wikipedia.org/wiki/Access_key#Access_in_different_Browsers)
